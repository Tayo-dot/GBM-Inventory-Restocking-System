# GBM-Based Inventory Restocking System

This project is a machine learning-based inventory restocking system developed using XGBoost, Flask, SQLite, and SQLAlchemy.

## Features
- Demand prediction using XGBoost
- Restocking recommendation logic
- Product management
- Stock management
- Restock history logging
- Batch CSV prediction

## Tech Stack
- Python
- Flask
- XGBoost
- Pandas
- Scikit-learn
- SQLite
- SQLAlchemy
